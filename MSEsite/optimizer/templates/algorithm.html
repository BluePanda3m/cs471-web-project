Gradient Descent

How It Works

Gradient Descent is an iterative first-order optimization algorithm. Starting from an initial guess $\theta^{(0)}$, it repeatedly moves in the direction of the steepest descent of the loss surface — i.e., opposite to the gradient.

$$ \theta^{(t+1)} = \theta^{(t)} - \alpha \cdot \nabla_\theta \mathcal{L}\!\left(\theta^{(t)}\right) $$

where $\alpha$ is the learning rate — a small positive scalar controlling the step size. We use $\alpha = 0.01$ and run for $500$ iterations.

Algorithm Steps

{% for step in "12345"|make_list %} {% endfor %}
  1. Initialize $\theta = \mathbf{0}$
  2. Compute predictions: $\hat{\mathbf{y}} = \mathbf{X}\theta$
  3. Compute MSE loss: $\mathcal{L} = \frac{1}{n}\|\mathbf{y}-\hat{\mathbf{y}}\|^2$
  4. Compute gradient: $g = \frac{2}{n}\mathbf{X}^\top(\hat{\mathbf{y}}-\mathbf{y})$
  5. Update parameters: $\theta \leftarrow \theta - \alpha g$
  6. Repeat steps 2–5 until convergence

Python Implementation

mse_algorithm.py 
import numpy as np

def compute_mse(y_true, y_pred):
    """MSE = (1/n) * sum((y - y_hat)^2)"""
    n = len(y_true)
    return (1 / n) * np.sum((y_true - y_pred) ** 2)


def compute_gradient(y_true, X, theta):
    """Gradient: (2/n) * X^T * (X*theta - y)"""
    n = len(y_true)
    y_pred = X @ theta
    grad   = (2 / n) * X.T @ (y_pred - y_true)
    return grad


def gradient_descent(X, y_true, learning_rate=0.01, iterations=500):
    theta        = np.zeros(X.shape[1])   # initialise at zero
    cost_history = []

    for _ in range(iterations):
        y_pred = X @ theta
        cost   = compute_mse(y_true, y_pred)
        cost_history.append(cost)

        grad  = compute_gradient(y_true, X, theta)
        theta = theta - learning_rate * grad  # update rule

    return theta, cost_history

Convergence Guarantee

Because MSE is strongly convex and the gradient is Lipschitz continuous, gradient descent with a sufficiently small learning rate is guaranteed to converge to the global optimum $\theta^*$. The convergence rate is linear:

$$ \mathcal{L}(\theta^{(t)}) - \mathcal{L}(\theta^*) \;\leq\; (1 - \alpha \mu)^t \left[\mathcal{L}(\theta^{(0)}) - \mathcal{L}(\theta^*)\right] $$

where $\mu$ is the strong-convexity constant. In practice, we observe rapid convergence within the first 100 iterations — see the Results page.

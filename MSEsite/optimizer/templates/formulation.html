Mathematical Formulation

4. Prediction Model

A simple linear regression model is used in this project. The prediction equation is:

$$ \hat{y} = wx + b $$
SymbolMeaning
$\hat{y}$Predicted value
$x$Input variable
$w$Weight (slope)
$b$Bias (intercept)

5. Decision Variables

Decision variables are the variables that can be adjusted during the optimization process in order to achieve the best solution. In this project, the decision variables are w and b.

  • w — controls the slope of the prediction line
  • b — controls the intercept of the prediction line

The optimization algorithm searches for the best values of w and b that minimize the prediction error.

6. Objective Function

The objective function used in this project is the Mean Squared Error (MSE). It measures the average squared difference between the actual values and the predicted values.

$$ \text{MSE}(w, b) = \frac{1}{n} \sum_{i=1}^{n} \left( y_i - \hat{y}_i \right)^2 $$
SymbolMeaning
$n$Number of data points
$y_i$Actual value
$\hat{y}_i$Predicted value
$(y_i - \hat{y}_i)$Prediction error

7. Why Mean Squared Error?

Mean Squared Error was selected because it is one of the most effective and widely used loss functions in regression problems. It gives higher penalties to large prediction errors because the error values are squared. This helps improve the accuracy of the model and makes the optimization process more stable.

In addition, MSE is mathematically simple and easy to implement using optimization algorithms such as Gradient Descent.

8. Mathematical Formulation

The prediction model is expressed as:

$$ \hat{y}_i = wx_i + b $$

The prediction error for each data point is:

$$ e_i = y_i - \hat{y}_i = y_i - (wx_i + b) $$

The Mean Squared Error becomes:

$$ \text{MSE}(w, b) = \frac{1}{n} \sum_{i=1}^{n} \left( y_i - (wx_i + b) \right)^2 $$

Therefore, the optimization problem is formulated as:

$$ \min_{w,\, b} \;\; \frac{1}{n} \sum_{i=1}^{n} \left( y_i - (wx_i + b) \right)^2 $$

Gradient of MSE

The partial derivatives used by gradient descent are:

$$ \frac{\partial \text{MSE}}{\partial w} = \frac{-2}{n} \sum_{i=1}^{n} x_i \left( y_i - \hat{y}_i \right) $$ $$ \frac{\partial \text{MSE}}{\partial b} = \frac{-2}{n} \sum_{i=1}^{n} \left( y_i - \hat{y}_i \right) $$

9. Constraints

This project represents an unconstrained optimization problem because there are no explicit restrictions on the values of the decision variables w and b. Therefore, the optimization algorithm is free to search for any values that minimize the objective function.

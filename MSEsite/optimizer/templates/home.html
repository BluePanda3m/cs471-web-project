Mean Squared Error Minimization

CS348 Course Project — Optimization

1. Introduction

Optimization is an important concept in mathematics, computer science, and machine learning. It is used to find the best possible solution for a problem by minimizing or maximizing a specific function. In machine learning applications, optimization is commonly used to reduce prediction errors and improve model performance.

In this project, we focus on minimizing the prediction error using the Mean Squared Error (MSE) function. MSE is one of the most widely used loss functions in regression problems because it measures the difference between the actual values and the predicted values accurately.

The main objective of this project is to formulate the optimization problem mathematically and find the values of the model parameters that minimize the error. This project also helps connect the theoretical understanding of optimization with practical implementation and visualization.

2. Optimization Idea

The main idea of optimization in this project is to minimize the difference between the predicted output and the actual output. A prediction model is used to estimate output values based on input data. However, the predicted values are usually not exactly equal to the actual values, which creates an error.

To improve the model performance, we use an objective function that measures this error and then try to minimize it. In this project, the Mean Squared Error (MSE) is used as the objective function. The optimization process searches for the best parameter values that produce the smallest possible error.

3. Problem Statement

Given a dataset containing input values and actual output values, the problem is to determine the optimal model parameters w (weight/slope) and b (bias/intercept) that minimize the prediction error.

The prediction model attempts to estimate the output values, while the optimization process adjusts the model parameters until the Mean Squared Error becomes as small as possible.

$$ \min_{w,\, b} \; \text{MSE}(w, b) = \frac{1}{n} \sum_{i=1}^{n} \left( y_i - (wx_i + b) \right)^2 $$
